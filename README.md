# University of Ibadan Enrolment Analytics Platform
 
## Business Problem
Universities struggle to predict student enrolment trends, leading to poor resource allocation
(hostels, staff, budgets).

## Objective
To build a data-driven analytics platform that predicts enrolment trends and supports
evidence-based decision-making.

## Results & Insights

- Highlighted staff-to-student imbalance risks
- Predicted enrolment trends with X% accuracy

## Business Impact
This solution can help university administrators:
- Plan staffing requirements
- Optimize hostel allocation
- Improve budget forecasting
